Class Lualog

Create a logger instance what you can customize how elements are printed.

Methods

Lualog:info (...) Log info

Methods

Lualog:new ([options={}])
Create a Lualog instance

Parameters:

  • options Options
    • datestring string or boolean Date string for os.date(). If false, date is not logged (default false)
    • tag string Logger tag. (default "")
    • ignore_levels {string,...} Array of levels ignored. (default {})
    • styles {} Table that define styles. (default {})
    • table_inspect table Table inspector config.
    • table_inspect.prettyfy boolean Flag to pretty printing tables. (default false)
    • table_inspect.allow_tostring boolean Flag to allow use table __tostring metamethod. (default true)
    • table_inspect.level_depth boolean Depth level to inspect tables. 0 means no level limit. (default 0)
    • plugins {function,...} Plugins. (default {})

Returns:

    Lualog
